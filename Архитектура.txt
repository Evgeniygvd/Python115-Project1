2. Глобальная переменная

Этапы работы приложения:

1. Приветствие

1. Чтение из ini-файла

    а) (если первый раз) вывод справки

2. Суперцикл для работы приложения: обработка команд

    а) предложение начать новую партию (или ввести другую команду)

    а) (если нет текущего игрока) запрос имени игрока

    б) предложение выбрать режим игры

        если одиночная:

            i) (если есть save) предложить загрузить сохраненную партию

            ii) (если нет save) выбор сложности

            iii) (если нет save) выбор символа

            iv) старт партии вложенный цикл для обработки ходов для партии

        если для двоих:

            i) запрос имени второго игрока

            ii) (если есть save) предложить загрузить сохраненную партию

            iii) выбор символа для первого игрока

            iv) старт партии
                вложенный цикл для обработки ввода ходов для партии

    д) вывод статистики для текущего (-их) игрока (-ов)

    е) предложение начать новую партию

3. Вложенный цикл для обработки ввода ходов для партии
